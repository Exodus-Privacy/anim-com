What we call a tracker

A tracker is a piece of software meant to collect data about you or your usages.







Like Ogury, Google Analytics, Teemo, and others.

How we detect them

Static analysis

  • List Java classes embedded in the APK
  • Find classes matching the tracker code signature
  • Dexdump, Androguard, Gplaycli

Network analysis - WIP

  • List DNS, HTTP/S, UDP requests
  • Find requests matching the tracker network signature
  • PyShark, Wireshark, MITMProxy, SSLStrip, Virtual Box
